## 📖 项目简介
人类文明正处于“反馈脱钩”的边缘。复杂的治理层屏蔽了真实的生存痛感，导致系统熵增（紊乱）不可逆转。**文明校正主义**是一套旨在重建文明自愈能力的开源框架。

我们不提供意识形态，我们提供**物理感知的防御协议**。

## 核心支柱 (The Pillars)
1. **本体反馈 (Ontological Feedback):** 个体（本体）是文明中唯一真实的传感器。
2. **感知协议 (CSP):** 一套去中心化的、保护隐私的底层协议，用于采集真实的“社会痛感”。
3. **理性辅助 (Rational Auxiliary):** 利用 AI 抹平信息差，辅助本体看清隐性系统成本，实现“知情反馈”。
4. **权力随机化 (Randomization):** 利用数学随机性对冲利益集团的合谋。

## 🛠 技术栈 (Technical Stack)
- **Core:** Rust (High performance, Memory safety)
- **Privacy:** Differential Privacy (OpenDP)
- **Network:** Libp2p (P2P decentralized network)
- **Verification:** RTAI (Real-time Truth & Authenticity Index)
