## 🌟 愿景 (The Mission)

在 2026 年，人类社会正面临严重的 **“系统漂移” (Systemic Drift)**：宏观经济指标、政策数据与个体的真实生存感受（本体体验）发生了严重的解耦。

**Civilizational Calibrationism (文明校正主义)** 提供了一套基于控制论和信息物理学的防御框架。我们不预设绝对的“正确答案”，我们只提供一套标定工具，利用 AI 辅助的本体感知，强制将偏离的行政执行重新对齐到个体的真实福祉之上。

## 🧠 核心支柱 (The Pillars)

1.  **本体反馈 (Ontological Feedback)**
    个体（本体）是文明中唯一真实的、不可替代的传感器。只有本体知道自己“疼不疼”。这种感知权是原生的主权。
2.  **文明感知协议 (CSP - Civilization Sensing Protocol)**
    一套去中心化、保护隐私的底层通信协议。捕捉分布式“社会痛感”，并在不泄露本体身份的前提下将其聚合成校正信号。
3.  **理性辅助 (Rational Auxiliary)**
    AI 被严格定义为 **“本体理性的感知外骨骼”**。负责穿透信息迷雾，将宏观成本翻译为本体可理解的生存账单。
4.  **权力随机化 (Randomization)**
    利用数学层面的随机抽样对冲利益集团合谋，确保审计权与决策权回归随机抽中的、受理性辅助的本体。

## ⚖️ 为什么是校正 (Calibration) 而非纠正 (Correction)？

我们放弃了带有威权色彩的 "Correction"，转而拥抱控制论背景的 **"Calibration"**：

- **非惩戒性**：校正不是审判，而是对齐。
- **动态持续**：系统始终在漂移，校正必须是实时的、高频的。
- **基准对齐**：以“本体生存体感”为基准（Reference Standard），消除执行层的偏差。
